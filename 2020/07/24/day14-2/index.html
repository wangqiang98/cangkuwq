User-Agent

User-Agent是存放在| Headers 中的一种数据信息。作用是,在指定 URL 发送请求的时候，告诉服务端当前用户的浏览器类型、版本，甚至操作系统、CPU等非隐私的技术信息。服务器从Headers 中的| User- Agent信息获取到浏览器类型、版本等数据后，就认为是一个浏览器请求的环境了，就会给出响应。

使用User-Agent

只需要在构建 Request 对象的时候，调用 addHeader() 方法即可

Request request = new Request.Builder()
.url(url)
.addHeader(“User-Agent”, “”)
.build();

addHeader（） 方法第一个参数是名称，第二个参数是值。

例题：

请求调用http，在 console 打印 http 响应状态码，以及最终响应的 URL。
