Response -网页

http状态码

2020表示成功，404表示出错

为了响应状态码，可以使用语句：call.execute().code()

但是通常还要读取响应内容，但是又不能执行两次请求，代码优化为：

import okhttp3.Response;

// 执行请求
Response rep = call.execute();
// 获取响应状态码
int code = rep.code();
// 获取响应内容
String content = rep.body().string();
