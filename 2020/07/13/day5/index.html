** 一个小偷沿街偷取现金，偷取相邻两家则会报警，在不触动报警情况下最多偷取多少现金？**

案例一

输入（房间依次现金）: [1,2,3,1]
输出: 4
解释: 偷窃 1 号房屋 (金额 = 1) ，然后偷窃 3 号房屋 (金额 = 3)。偷窃到的最高金额 = 1 + 3 = 4 。

案例二

输入: [2,7,9,3,1]
输出: 12
解释: 偷窃 1 号房屋 (金额 = 2), 偷窃 3 号房屋 (金额 = 9)，接着偷窃 5 号房屋 (金额 = 1)。
偷窃到的最高金额 = 2 + 9 + 1 = 12 。

代码

import java.util.HashMap;

public class Demo {

public static HashMap<Integer, Integer> cache = new HashMap<>();
public static int f(int[] rooms) {
    // 每次调用的时候，将cache情况
    cache.clear();
    return fCore(rooms, rooms.length - 1);
}

public static int fCore(int[] rooms, int index) {
    // 处理边界情况
    if (index < 0) {
        return 0;
    }
    if (index == 0) {
        return rooms[index];
    }
    if (cache.containsKey(index)) {
        return cache.get(index);
    }
    // 最后一个房间偷或者不偷两种情况。
    int max = Math.max(fCore(rooms, index - 1), fCore(rooms, index - 2) + rooms[index]);
    cache.put(index, max);
    return max;
}
public static void main(String[] args) {
    int[] rooms1 = {1, 2, 3, 1};
    System.out.println("{1,2,3,1}" + f(rooms1));
    int[] rooms2 = {2, 7, 9, 3, 1};
    System.out.println("{2,7,9,3,1}" + f(rooms2));

}

}
